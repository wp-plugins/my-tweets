== Description ==

This Twitter plugin allows you to publish your latest twitter updates as a widget on your blog.

You can select how many updates you wish to publish on your website, including options to make URLs into hyperlinks, replies into hyperlinks and even showing your profile picture!

== Installation ==

1. Upload `my-tweets.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit the "My Tweets" Settings in your Administration Panel
4. Enable the Widget through the Appearance --> Widgets tab.
